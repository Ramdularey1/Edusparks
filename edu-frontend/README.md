# Edu Frontend

Scaffolded Vite + React (JavaScript) project for the EduSpark frontend.

Quick start

```bash
cd edu-frontend
npm install
npm run dev
```

Notes
- This scaffold includes basic routing, a header, sample pages, an `api` axios client, and a simple `useAuth` hook.
- To add Tailwind, run the Tailwind install steps and update `src/index.css` per docs.
